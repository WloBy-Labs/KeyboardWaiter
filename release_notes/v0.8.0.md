# KeyboardWaiter 0.8.0

- Added a dedicated app icon (graphite background with a keyboard and mouse).
- Added a GitHub Actions release pipeline that builds the app and publishes a DMG on every version tag.
- Distributed the app as a drag-to-install DMG with an Applications shortcut.
- Added progressive code signing: ad-hoc by default, a stable certificate when configured, and notarization when Apple credentials are provided.
- Signed releases with a stable identity so Input Monitoring permission is kept across updates.
